Grievance Portal

Each department has a separate view of the Grievance portal where they can view the list of complaints. As soon as there is an update in status, the individual should be able to view it . A user can post his/her complaint along with photos/document to support their claim. He/she should also select the department the complaint should be assigned to. (Rather than user selecting the department, look into classification techniques to assign the department). After posting the complaint, the user can view the current status of the complaint.
